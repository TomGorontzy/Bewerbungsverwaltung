# SCHNELLSTART

Diese Anleitung zeigt nur die **konkreten Schritte**, die im Alltag funktionieren.

> Hinweis: Die Pflege der Bewerbungsaktivitäten kann sowohl über die Anwendung als auch direkt über die Microsoft Excel-Arbeitsmappe im Tabellenblatt `Bewerbungsübersicht` erfolgen.

## A) Arbeit mit der Anwendung

### 1) Vorbereiten

1. Laden Sie die aktuelle Version als ZIP-Archiv auf Ihren Rechner herunter und extrahieren Sie die Inhalte.
2. Arbeiten Sie mit den Inhalten dieses Ordners weiter.
3. Schließen Sie die Excel-Datei, falls sie geöffnet ist.

### 2) Anwendung starten

1. Starten Sie `Bewerbungsverwaltung.exe` per Doppelklick.
2. Falls Windows eine Warnung zeigt: `Weitere Informationen` → `Trotzdem ausführen`.
3. Warten Sie kurz, bis die Daten geladen sind.

### 3) Neue Bewerbung erfassen

1. Wechseln Sie in den Tab **Neue Bewerbungen**.
2. Füllen Sie mindestens diese Pflichtfelder aus:
   - Unternehmen
   - Position
   - Datum der Bewerbung
   - Aktueller Status
3. Tragen Sie optional weitere Felder ein (z. B. `Nächster Schritt`, `Priorität`, `Notizen`).
4. Klicken Sie auf **Speichern**.

### 4) Nachfassen erledigen

1. Wechseln Sie in den Tab **Nachzufassende Bewerbungen**.
2. Nutzen Sie bei Bedarf das Feld **Filter**.
3. Wählen Sie einen Eintrag in der Liste aus.
4. Aktualisieren Sie unten die Felder (z. B. `Neuer Status`, `Kontaktart`, `Nächster Schritt`).
5. Klicken Sie auf **Änderung speichern**.

## B) Direkte Bearbeitung in Microsoft Excel

### 1) Vorbereiten (2)

1. Laden Sie die aktuelle Version als ZIP-Archiv auf Ihren Rechner herunter und extrahieren Sie die Inhalte.
2. Arbeiten Sie mit den Inhalten dieses Ordners weiter.
3. Öffnen Sie `data/Bewerbungsaktivitäten mit Erinnerungen.xlsx` in Microsoft Excel.

### 2) Excel-Datei öffnen und Blatt wählen

1. Öffnen Sie die Arbeitsmappe `data/Bewerbungsaktivitäten mit Erinnerungen.xlsx`.
2. Wechseln Sie in das Tabellenblatt **Bewerbungsübersicht**.
3. Nutzen Sie die vorhandenen Dropdowns in den relevanten Spalten für einheitliche Werte.

### 3) Neue Bewerbung erfassen (2)

1. Wechseln Sie in die nächste freie Zeile im Blatt **Bewerbungsübersicht**.
2. Tragen Sie mindestens diese Felder ein:
   - Unternehmen (Spalte B)
   - Position (Spalte D)
   - Datum der Bewerbung (Spalte J)
   - Aktueller Status (Spalte L)
3. Ergänzen Sie optional weitere Felder (z. B. `Nächster Schritt` in Spalte Q, `Priorität` in Spalte S, `Notizen` in Spalte U).
4. Speichern Sie die Arbeitsmappe.

### 4) Nachfassen erledigen (2)

1. Bleiben Sie im Blatt **Bewerbungsübersicht** und suchen Sie den relevanten Eintrag.
2. Aktualisieren Sie die Nachfassungsfelder (z. B. `Status` in Spalte L, `Status-Datum` in Spalte M, `Art Kontaktaufnahme` in Spalte O, `Nächster Schritt` in Spalte Q).
3. Prüfen Sie bei Bedarf das daraus abgeleitete `Erinnerungsdatum` in Spalte R.
4. Speichern Sie die Arbeitsmappe.

> Hinweis: Nachzufassende Bewerbungen werden in Excel automatisch in den Tabellenblättern **Heute erledigen** und **Diese Woche erledigen** angezeigt.

## 5) Farben richtig lesen

Im Tab **Nachzufassende Bewerbungen** gilt:

- hellgelb = 1–2 Tage überfällig
- orange-beige = 3–6 Tage überfällig
- orange = 7–13 Tage überfällig
- hellrot = ab 14 Tagen überfällig

Die gleiche Einordnung steht auch in der Legende im Tab.

## 6) Katalog „Nächster Schritt“ pflegen

1. Klicken Sie im Tab **Neue Bewerbungen** auf **Schritt-Katalog verwalten**.
2. Fügen Sie Einträge hinzu, löschen Sie Einträge oder ändern Sie die Reihenfolge.
3. Klicken Sie auf **Speichern**.

Die Änderungen stehen danach direkt in den Dropdowns zur Verfügung.

## 7) Tagesroutine (empfohlen)

1. Starten Sie die Anwendung.
2. Öffnen Sie den Tab **Nachzufassende Bewerbungen**.
3. Bearbeiten Sie überfällige Einträge (orange/rot) zuerst.
4. Speichern Sie die Änderungen.
5. Erfassen Sie optional neue Bewerbungen.

## 8) Wenn Speichern nicht klappt

1. Schließen Sie die Excel-Datei.
2. Warten Sie 5–10 Sekunden (OneDrive-Synchronisierung).
3. Speichern Sie erneut.
4. Klicken Sie bei Bedarf auf **Daten neu laden**.
